The following table describes the types of events that are native to Centrapay. No actions is required as the native events are handled by Centrapay.


|      Event Type       |                            Category                            |
| :-------------------- | :------------------------------------------------------------- |
| `refund:completed`    | [Payment Request](/api/payment-requests#payment-request-model) |
| `promotion:completed` | [Loyalty Program](/api/loyalty-programs#loyalty-program-model) |
| `promotion:updated`   | [Loyalty Program](/api/loyalty-programs#loyalty-program-model) |
